# Convert grobid to Bibr format

Convert grobid to Bibr format

## Usage

``` r
.grobid_to_bibr(xml_path, pb = NULL)
```

## Arguments

- xml_path:

  path to a single XML file

- pb:

  a progress bar passed from
  [`grobid_to_bibr()`](https://scienceverse.github.io/metacheck/reference/grobid_to_bibr.md)

## Value

a paper object
