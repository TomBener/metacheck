# Launch Report App

Launch the Report app: upload a PDF and generate a report with one
click, with privacy options for what is sent to external servers.

## Usage

``` r
report_app(quiet = FALSE, ...)
```

## Arguments

- quiet:

  whether to show debugging messages in the console

- ...:

  arguments to pass to shiny::runApp

## Value

NULL (invisibly)

## Examples

``` r
if (FALSE) { # \dontrun{
report_app()
} # }
```
