# Batch query

Batch query

## Usage

``` r
.batch_query(urls, batch_size = 5, msg = "Batch Query", delay = 0.5)
```

## Arguments

- urls:

  A vector of URLs

- batch_size:

  Size of each batch

- msg:

  Message to show in progress bar

- delay:

  Courtesy delay between batches (in seconds)

## Value

a list of responses
