# View a figure image

View a figure image

## Usage

``` r
fig_image_view(paper, figure_id = 1)
```

## Arguments

- paper:

  a paper object

- figure_id:

  the id for the figure to show

## Value

plots the figure

## Examples

``` r
paper <- demopaper()
fig_image_view(paper, 1)
#> NULL
fig_image_view(paper, 2)
#> NULL
```
