# Create Report from Module Output

Create Report from Module Output

## Usage

``` r
report_qmd(module_output, paper = list())
```

## Arguments

- module_output:

  a list of module output (usually from
  [`report_module_run()`](https://scienceverse.github.io/metacheck/dev/reference/report_module_run.md))

- paper:

  a paper object

## Value

report text
