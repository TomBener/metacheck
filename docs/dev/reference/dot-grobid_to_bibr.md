# Convert grobid to Bibr format

Convert grobid to Bibr format

## Usage

``` r
.grobid_to_bibr(xml_file, pb = NULL)
```

## Arguments

- xml_file:

  a singhle XML file

- pb:

  a progress bar passed from
  [`grobid_to_bibr()`](https://scienceverse.github.io/metacheck/dev/reference/grobid_to_bibr.md)

## Value

a paper object
