# List Files in an OSF Component

List Files in an OSF Component

## Usage

``` r
osf_files(osf_id, osf_type = NULL, pb = NULL)
```

## Arguments

- osf_id:

  an OSF ID

- osf_type:

  the osf component type (usually nodes, registrations, or files)

- pb:

  a progress bar passed from another function

## Value

a data frame with file info
