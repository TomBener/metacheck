# Get README from GitHub

Get README from GitHub

## Usage

``` r
github_readme(repo)
```

## Arguments

- repo:

  The URL of the repository (in the format "username/repo" or
  "https://github.com/username/repo")

## Value

a character string of the README contents

## Examples

``` r
if (FALSE) { # \dontrun{
github_readme("scienceverse/metacheck")
} # }
```
