# Get Languages from GitHub Repo

Get Languages from GitHub Repo

## Usage

``` r
github_languages(repo)
```

## Arguments

- repo:

  The URL of the repository (in the format "username/repo" or
  "https://github.com/username/repo")

## Value

vector of languages

## Examples

``` r
if (FALSE) { # \dontrun{
github_languages("scienceverse/metacheck")
} # }
```
